========================================================
  מונדיאל 2026 — עדכוני תוצאות בלייב (שרת מקומי)
========================================================

מה זה: שרת קטן שרץ אצלך במחשב, מושך תוצאות אמיתיות
מהאינטרנט כל דקה, ומעדכן את האתר אוטומטית — כולל
משחקים שמסתיימים בזמן אמת.

------------------------------------------------------
הגדרה חד-פעמית (5 דקות)
------------------------------------------------------

1) קבל מפתח API חינמי:
   • היכנס ל-  https://www.football-data.org/client/register
   • הירשם עם אימייל (חינם, בלי כרטיס אשראי)
   • תקבל באימייל "API Token" — מחרוזת אותיות ומספרים

2) הדבק את המפתח:
   • פתח את הקובץ  api-key.txt  (נמצא כאן בתיקייה)
   • הדבק לתוכו את המפתח בלבד, שמור וסגור

3) הפעל את השרת:
   • פתח חלון PowerShell בתיקייה הזו
     (לחיצה ימנית בתיקייה -> "Open in Terminal")
   • הקלד:   node live-server.js
   • אמור להופיע "LIVE server is running"

4) פתח בדפדפן:
   • היכנס ל-  http://localhost:8080
   • זהו! התוצאות יתעדכנו לבד כל דקה.
     (תוכל לראות את הסטטוס בטאב "ניהול נתונים")

------------------------------------------------------
שימוש יומיומי
------------------------------------------------------

בכל פעם שתרצה לעבוד עם האתר:
  1. node live-server.js
  2. http://localhost:8080

לעצירת השרת: Ctrl+C בחלון הטרמינל.

------------------------------------------------------
הערות
------------------------------------------------------

• בלי השרת, אפשר עדיין לפתוח את index.html ישירות —
  פשוט בלי עדכון אוטומטי.
• אם המפתח חסר/שגוי, האתר ייפול אוטומטית למקור גיבוי
  (TheSportsDB) ויציג הודעה בטאב "ניהול נתונים".
• football-data.org במסלול החינמי מכסה את המונדיאל.
  אם בכל זאת תיחסם גישה — כתוב לי ונתאים מקור אחר.
